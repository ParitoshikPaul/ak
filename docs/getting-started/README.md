# Ambika Kapoor Makeup Studio Website

A beautiful, SEO-optimized one-page website for a professional makeup artist based in Srinagar, designed with pastel makeup theme colors.

## Features

- **SEO Optimized** for Srinagar local searches
- **Responsive Design** - works on all devices
- **Pastel Makeup Color Theme** - soft pinks, peaches, lavenders
- **Smooth Animations** and transitions
- **Contact Form** for booking appointments
- **Portfolio Gallery** section
- **Client Testimonials** slider
- **Services Showcase**
- **Social Media Integration**

## Files Included

- `index.html` - Main HTML file with SEO meta tags and schema markup
- `styles.css` - Complete styling with pastel color palette
- `script.js` - Interactive features and animations
- `README.md` - This file

## How to Use

### 1. Customize Content

#### Update Contact Information
In `index.html`, replace these placeholders:
- Email: Change `info@ambikakapoor.com` to your actual email
- Phone: Replace `+91 XXXXX XXXXX` with your phone number
- Address: Update location details with your actual address

#### Add Real Photos
Replace the placeholder sections with your actual photos:
1. About section photo (line ~151)
2. Portfolio images (lines ~287-344)
3. Add your logo/favicon

#### Update Social Media Links
- Facebook: Already linked to your profile
- Instagram: Add your Instagram handle
- WhatsApp: Update phone number in `script.js` (line 136)

### 2. SEO Optimization Steps

#### Local SEO for Srinagar

1. **Google My Business**
   - Create/claim your Google My Business listing
   - Use the same business name: "Ambika Kapoor Makeup Studio"
   - Add your Srinagar address
   - Upload portfolio photos
   - Collect reviews from clients

2. **Schema Markup**
   - Already included in the HTML (lines 30-89)
   - Update with your actual:
     - Phone number
     - Street address
     - Postal code
     - Business hours

3. **Meta Tags**
   - Already optimized for keywords:
     - "bridal makeup artist Srinagar"
     - "party makeup artist Srinagar"
     - "freelance makeup artist Kashmir"
   - These are in the `<head>` section

4. **Content Keywords**
   - Keywords already integrated naturally throughout the page
   - "Srinagar" mentioned 15+ times
   - Service-specific keywords included

#### Get on Google Search Results

1. **Submit to Google Search Console**
   ```
   - Go to: https://search.google.com/search-console
   - Add property: https://www.ambikakapoor.com
   - Verify ownership
   - Submit sitemap
   ```

2. **Create Social Signals**
   - Post regularly on Facebook with location tags
   - Use Instagram with Srinagar location
   - Encourage clients to tag you in posts

3. **Get Backlinks**
   - List on wedding directories
   - Partner with wedding venues in Srinagar
   - Get featured in local blogs/magazines

4. **Collect Reviews**
   - Ask happy clients for Google reviews
   - Share testimonials with photos
   - Respond to all reviews

### 3. Additional SEO Files to Create

#### Create sitemap.xml
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://www.ambikakapoor.com/</loc>
    <lastmod>2024-01-01</lastmod>
    <priority>1.0</priority>
  </url>
</urlset>
```

#### Create robots.txt
```
User-agent: *
Allow: /
Sitemap: https://www.ambikakapoor.com/sitemap.xml
```

### 4. Deployment

#### Option 1: GitHub Pages (Free)
1. Create a GitHub account
2. Create repository named `ambikakapoor.com`
3. Upload files
4. Enable GitHub Pages in settings
5. Connect custom domain

#### Option 2: Netlify (Free)
1. Sign up at netlify.com
2. Drag and drop your folder
3. Connect custom domain
4. Site goes live instantly

#### Option 3: Traditional Hosting
1. Purchase hosting (Hostinger, Bluehost, etc.)
2. Upload files via FTP
3. Point domain to hosting
