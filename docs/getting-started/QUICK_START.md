# Quick Start Guide

## Getting Started in 5 Minutes

### Step 1: Open the Website
Simply open `index.html` in your web browser to see your website.

### Step 2: Customize These Essential Items

1. **Your Contact Info** (in `index.html`)
   - Line 152: Your email address
   - Line 157: Your phone number
   - Line 163: Your full address in Srinagar
   - Line 176: Update Facebook link if different

2. **Your Photos**
   - Replace placeholder sections with real photos
   - Recommended: Add professional headshot in About section
   - Add 6 portfolio photos in Portfolio section

### Step 3: Test the Website
- Click all navigation links
- Test the contact form
- Check on mobile (use browser's responsive mode)

### Step 4: Deploy Online

**Easiest Method - Netlify (5 minutes):**

1. Go to [netlify.com](https://netlify.com)
2. Sign up for free account
3. Drag and drop your `ak` folder
4. Your site is live at a free URL like: `yoursite.netlify.app`
5. (Optional) Connect your domain `ambikakapoor.com`

**Alternative - GitHub Pages:**

1. Create account at [github.com](https://github.com)
2. Create new repository
3. Upload files
4. Enable GitHub Pages in Settings

### Step 5: Get Found on Google

1. **Google My Business** (Most Important!)
   - Go to [business.google.com](https://business.google.com)
   - Create listing for "Ambika Kapoor Makeup Studio"
   - Add your Srinagar address
   - Upload photos
   - This gets you on Google Maps!

2. **Google Search Console**
   - Go to [search.google.com/search-console](https://search.google.com/search-console)
   - Add your website
   - Submit sitemap: `yoursite.com/sitemap.xml`

3. **Social Media**
   - Add website link to Facebook page
   - Create Instagram business profile
   - Add website link to WhatsApp Business profile

### Step 6: Start Getting Clients

1. Share website on your social media
2. Add link to your Facebook videos
3. Include in your Instagram bio
4. Ask happy clients for Google reviews
5. Share before/after photos with website link

## Need Help?

Common tasks:

**Change Colors:**
- Open `styles.css`
- Look for `:root` section at top
- Change color values there

**Add More Services:**
- Open `index.html`
- Find Services section (around line 220)
- Copy a service card and modify

**Update Testimonials:**
- Open `index.html`
- Find Testimonials section (around line 350)
- Replace the text with real client reviews

**Change Phone/Email:**
- Search in `index.html` for "XXXXX"
- Replace all instances
